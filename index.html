Спецефичность селекторов

Составные селектора

  • груповые селектора

    Например

    • h1, h2, span { color:red }
  • Селектор потомок или контекстный селектор
    • ul li code {color blue} стили будут применены к последнму (дочерниму) елементу этой цепочки
  • Селектор прямого потомка
    • ul li > span {font-weight:bolt} стили будут применены к span (дочерниму) елементу этой цепочки
  • Соседний селектор или следующий
    • ul li + li { } > span {font-weight:bolt} стили будут применены к следующему селектору после указанного
